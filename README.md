To run the program:

# Frontend:

1) npm install
2) npm start


# Backend:
1) Edit .env file:
Plaid Configuration
PLAID_CLIENT_ID=your_plaid_client_id
PLAID_SECRET=your_plaid_secret
PLAID_ENV=sandbox

Database Configuration
DB_SERVER=localhost
DB_DATABASE=personalFinance
DB_USER=sa
DB_PASSWORD=your_db_password

2) node
3) node app.js
